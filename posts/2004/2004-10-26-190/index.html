Blog for nightly build changelog

October 26, 2004 development

The Burning Edge 是 Mozilla Firefox 用來追蹤 nightly build 狀態的 blog，算是一種蠻特別的 blog 應用。在裡面，你可以看到每天 bug 修正的進度以及當天 build 的狀況，例如：

  • Fixed: 265839 - External protocol handling broken (since Oct 21).
  • Fixed: 263846 - Crash [@ nsImageFrame::FrameChanged][@ nsImageFrame::SourceRectToDest].
  • Fixed: 265849 - There’s no longer an Options button in the Download Manager.
  • Fixed: 265826 - Update linux installer artwork.
  • Fixed: 265829 - History code passes incorrect length to NS_ConvertUTF16toUTF8.

Official Windows, Official Windows installer (discussion), bangbang’s

Official Linux, Official Linux installer, polaris’

Official Mac, PowerBook’s

本來我以為這樣的內容是由程式自動產生的，但看起來有些不規律，似乎是人工新增上去的？ 如果是自動產生的，那實在是很不錯，至少 release note 只要剪剪貼貼就可以搞定了。
